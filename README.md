Box0.jl
=======

Julia Binding for libbox0

Dependencies
============

Actually: None

* libbox0 (C library)

Installation
============

The package is registered in `METADATA.jl` and can be installed with `Pkg.add`.

```
julia> Pkg.add("Box0")
```
